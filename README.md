# master-thesis-oilspill
Context-aware oil spill detection in Sentinel-1 SAR imagery using pretrained foundation model embeddings and scene-level CNNs.


## Repository status

This repository contains the core code used in the experiments presented in the master's thesis.

The codebase is currently being cleaned and documented. Additional scripts, evaluation utilities, and configuration files may be added in future updates.

The repository is provided primarily to illustrate the methodology and experimental pipeline described in the thesis.
